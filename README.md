## User Stories/BDD
These are the behaviours/features that the application implements for use by a user.

As a user I would like:
* To add new contact details - add your information
* Store my existing details

## Specifications
| Behaviour | Input | Output |
| :---------------- | :---------------: | ------------------: |
| Display codes for navigation | **In terminal: $./run.py** | Hello Welcome to your contact list. What is your name? |
| Display prompt for creating a new contact | **Enter: cc** | Enter your first name, last name, phone number and email address |
| Display a list of contacts | **Enter: dc** | Prints a list of saved contacts |
| Exit application | **Enter: ex** | Exit the current navigation stage |
